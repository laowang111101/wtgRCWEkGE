# 基于SpringBoot的幼儿园管理系统 java802

## 项目概述

本项目是基于SpringBoot框架，结合MyBatis持久层框架和Layui前端框架开发的幼儿园管理系统。系统实现了对幼儿园日常运营所需各项功能的管理，包括用户、页面、角色、教师、工资、物资、菜谱、班级、学生、公告、课程以及考勤等方面的管理。

## 技术栈

- **后端框架**：SpringBoot
- **持久层框架**：MyBatis
- **前端框架**：Layui
- **开发工具**：IntelliJ IDEA 2021.3
- **数据库**：Mysql 5.7.26
- **应用服务器**：Tomcat 7.0.73
- **JDK版本**：1.8

## 功能模块

### 系统管理

- 用户管理
- 页面管理
- 角色管理

### 校园管理

- 老师管理
- 工资管理
- 物资管理
- 菜谱管理
- 班级管理

### 班级管理

- 学生管理
- 公告管理
- 课程管理

### 考勤管理

- 老师考勤
- 学生考勤
- 老师考勤统计
- 学生考勤统计
- 签到签退

## 系统角色

系统主要角色包括系统管理员、教师、学生等，各角色根据权限不同，可进行不同的操作。

## 运行环境

- JDK 1.8 或更高版本
- Mysql 5.7.26 或更高版本
- Tomcat 7.0.73 或更高版本

## 部署步骤

1. 准备运行环境（JDK、Mysql、Tomcat）
2. 创建数据库，导入数据表结构
3. 部署应用到Tomcat服务器
4. 启动Tomcat，访问系统
